## 🔍 About ZorkOS-Termux  

ZorkOS-Termux is a tool that lets you change how your Termux terminal looks and works. It adds themes, animations, special hacker modes, and even achievements. You get over 17 extra features or modules. The goal is to make your terminal simple to use and fun to explore, even if you are not a tech expert.

This software works inside Termux, a terminal app on Android. If you want to customize or improve your terminal experience, this is what you need.

## 🚀 Getting Started  

To use ZorkOS-Termux, you first need Termux installed on your Android device. Termux acts as a basic Linux terminal emulator app for Android.

### Step 1: Install Termux  

1. Open the Google Play Store or F-Droid app on your Android device.  
2. Search for "Termux" by Fredrik Fornwall.  
3. Tap "Install" and wait for the app to finish installing.

## 💻 Running ZorkOS-Termux Setup  

Once you have Termux and the installer downloaded, follow these steps to get ZorkOS-Termux running.

### Step 4: Open Termux  

1. Launch the Termux app on your Android device.  
2. Termux will open a terminal screen with a cursor waiting for commands.

### Step 5: Move to the Download Folder  

Type this command and press Enter:  

```
cd ~/storage/downloads
```

This changes the current folder to where your downloaded files are saved.

*If you have not granted Termux permission to access storage, run:*

```
termux-setup-storage
```

Follow the prompts to allow permission, then try `cd ~/storage/downloads` again.

### Step 6: Make Installer File Executable  

Type this command:  

```
chmod +x ./[installer-file-name].sh
```

Replace `[installer-file-name]` with the actual file name you downloaded. This command lets you run the file as a program.

### Step 7: Run the Installer  

Now run the installer:  

```
./[installer-file-name].sh
```

Follow the instructions the installer shows. It will set up themes, animations, and other features automatically.

## 🔧 Basic Commands for ZorkOS-Termux  

Here are some simple commands you can use after installation.

- `zorkos-theme [theme-name]` — Change the theme to the name you specify.  
- `zorkos-hacker on/off` — Turn the hacker mode on or off.  
- `zorkos-achievements` — View your unlocked achievements.  
- `zorkos-modules list` — See all available modules.  
- `zorkos-modules enable [module-name]` — Activate a module.
